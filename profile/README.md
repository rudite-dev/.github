# Rudite

**Built in Pieces. Run as One.**

Rudite는 마인크래프트 서버를 위한 오픈소스 플러그인과 모듈형 시스템을 제작하는 개발 조직입니다.

서버 운영자가 더 쉽게 기능을 확장하고, 개발자가 더 안정적으로 시스템을 구성할 수 있도록 작고 명확한 도구들을 만듭니다.

## What we build

- 마인크래프트 서버 플러그인
- 서버 운영을 위한 유틸리티
- 게임플레이 확장 시스템
- 모듈형 서버 기능
- 개발자와 운영자를 위한 오픈소스 도구
- 서버 맞춤형 플러그인 커미션

## Philosophy

Rudite의 프로젝트는 하나의 거대한 시스템보다, 작고 독립적인 조각들이 함께 작동하는 구조를 지향합니다.

각 플러그인은 단순하고 명확하게 만들고, 필요할 때 서로 연결되어 하나의 서버 경험을 완성할 수 있도록 설계합니다.

## Open Source

Rudite의 공개 프로젝트는 자유롭게 사용하고, 수정하고, 참고할 수 있습니다.

우리는 서버 운영자와 개발자가 더 나은 서버 환경을 만들 수 있도록 실용적인 도구를 오픈소스로 제공합니다.

## Commissions

Rudite는 서버에 필요한 맞춤형 플러그인 제작 커미션도 받고 있습니다.

기존 플러그인으로 구현하기 어려운 기능, 서버만의 특수한 시스템, 미니게임, 운영 자동화 도구 등이 필요하다면 커미션을 통해 제작을 의뢰할 수 있습니다.

커미션 가능 작업은 프로젝트 규모, 요구사항, 서버 버전, 개발 난이도에 따라 검토 후 결정됩니다.

## Support

Rudite의 개발을 응원하고 싶다면 후원을 통해 프로젝트 유지보수와 새로운 기능 개발을 도울 수 있습니다.

> 조각으로 만들고, 하나처럼 움직입니다.
